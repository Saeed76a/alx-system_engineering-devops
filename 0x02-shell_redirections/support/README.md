# Local config

`This is a local tool that creates a bash file then opens the file in vi editor`

# Installation

`Run`

```bash
sudo ./config.ssh
```

# Usage

`To create a file: run`

```bash
create filename
```

`This makes the file created executable and opens vi with the following content appended in line 1`

```vi
#!/bin/bash

~
~
~
~
~
```
